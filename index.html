שאלות נפוצות על JavaScript

מהי שפת JavaScript?

JavaScript היא שפת תכנות מונחית עצמים המשמשת בעיקר לפיתוח צד לקוח (Frontend), ומשמשת להוספת אינטראקטיביות, אנימציות, בדיקות ולוגיקה לדפי אינטרנט.

מה ההבדל בין JavaScript ל־Java?

למרות הדמיון בשם, Java ו־JavaScript הן שפות שונות לחלוטין. Java היא שפה מונחית עצמים שמתקמפלת ומופעלת על שרתים, ואילו JavaScript רצה בדפדפן ומשמשת בעיקר לפיתוח אתרים.

האם אפשר להשתמש ב־JavaScript גם בצד השרת?

כן. באמצעות סביבת הריצה Node.js ניתן להריץ קוד JavaScript בצד השרת, לבנות API-ים, לנהל מסדי נתונים ועוד.

מהם משתנים ומה ההבדל בין var, let ו־const?

משתנים מאחסנים מידע. var הוא ישן ונפוץ פחות, let מאפשר שינוי ערך, ו־const מגדיר ערך קבוע שלא ניתן לשנות.

איך JavaScript מגיבה לאירועים בדף?

באמצעות Event Listeners – ניתן לקבוע פעולות שיתרחשו כשמתבצע אירוע כמו לחיצה, הקלדה או תנועה של העכבר. לדוגמה: element.addEventListener('click', function() {...})
